**Status bij schrijven:** main = `dd7ca0ff` (1.79.0 + CI-smoke-fix). PR #309 in merge-flow (checks draaien). Fixes #307 en #308 staan klaar op worktree-branches.

## Fase 5 — macOS-notarisatie (parallel spoor)

- Apple's notary-service geeft sinds 15-07 HTTP 500 (4× retry met backoff faalt ook).
- [ ] Bij release-build: slaagt notarisatie → klaar. Faalt hij opnieuw →
      gesigneerde-maar-niet-genotariseerde .dmg als asset toevoegen (rechtsklik →
      Openen bij eerste start) en dat in de release-notes vermelden
- [ ] Apple-accountstatus controleren op developer.apple.com (5 dagen storing is
      verdacht lang; mogelijk account-specifiek) en daarna notarisatie opnieuw
      draaien + asset vervangen

## Rollback

Release is een tag + GitHub-release: bij een ernstige regressie de release op
prerelease/draft zetten, fix op main, `v1.80.1` taggen. Geen force-pushes.
